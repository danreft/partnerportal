# Partner Portal

A Next.js application built with React, Ant Design, and Tailwind CSS for managing partner activities, referrals, and resources.

## Features

- User authentication and authorization
- Referral management
- Activity tracking
- Resource access
- Manager dashboard

## Installation

Ensure you have Node.js and pnpm installed.

```bash
pnpm install
```

## Development

Start the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

Build the application for production:

```bash
pnpm build
```

## Start

Start the production server:

```bash
pnpm start
```

## Linting

Run ESLint to check for code issues:

```bash
pnpm lint
```

## Technologies

- Next.js 15
- React 19
- Ant Design
- Tailwind CSS
- TypeScript
